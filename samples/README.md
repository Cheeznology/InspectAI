# InspectAI sample files

Use these to try **InspectAI: Review Selection** before reviewing your own code.

| File | Purpose |
| --- | --- |
| `01-clean-example.ts` | Solid TypeScript — expect a low risk score and few issues |
| `02-buggy-example.ts` | Intentional bugs — expect security, logic, and safety warnings |

## Quick test

1. Open a sample file.
2. Select a function (or select all).
3. Right-click → **InspectAI: Review Selection**.

These files live in the GitHub repo for demos and screenshots. They are not bundled inside the published VS Code extension.
